# Lens2CAD Backend

Backend service for Lens2CAD, powered by FastAPI, OpenCV, and U2Net.

## API Endpoints

- `GET /health`: Health check
- `POST /upload`: Upload image for processing

## Deployment

This service is designed to run on Hugging Face Spaces (Docker).
